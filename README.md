# Auburn High School MA Intro Page

This is a static HTML intro page.

## Preview locally

From the repo root:

```bash
python -m http.server 8000
```

Then open:

```
http://127.0.0.1:8000/index.html
```

You can also open `index.html` directly in a browser if you prefer.
